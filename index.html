Fórmula: factor = (480 − max(0, ida+vuelta − 60)) / 480

Sincronización automática — GitHub Gist
☁ Con el token configurado, cada día guardado se sube automáticamente al Gist y se descarga al abrir la app en cualquier dispositivo. Así PC y móvil siempre están sincronizados.

Cómo obtener el token: github.com → Settings → Developer settings → Personal access tokens → Tokens (classic) → New token → marcar gist → Generate.
Sincronización Google Drive
☁ Guarda y recupera tus datos directamente en tu Google Drive. Una vez conectado, cada día guardado se sube automáticamente y se descarga al abrir la app en cualquier dispositivo.
