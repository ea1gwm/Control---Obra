☁️ Sincronización en la Nube (GitHub Gist)

Guarda y recupera tus datos desde cualquier dispositivo usando un GitHub Gist privado. Necesitas una cuenta gratuita en GitHub.

1. Ve a github.com/settings/tokens → genera un token con permiso gist
2. Pega el token aquí. La primera vez se crea el Gist automáticamente.
3. Guarda el Gist ID que aparece — introdúcelo en otros dispositivos para sincronizar.
